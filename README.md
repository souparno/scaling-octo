social-mini
===========

a mini social networking site around panoramio. A user can register and login through facebook and find photos anywhere in the world and comment on a photo

features
========

Streams
=======
o	 Recent<br/>
o	 Popular<br/>
o	 Nearby<br/>  
o	 Trending<br/> 
o	 User Activity<br/>   
o	 My activity<br/> 
User (Subjective)
=================
o	Like<br/>
o	Flag<br/>
o	Comment<br/>
o	Tag cloud<br/>
o	Suggestions<br/>



Technologies 
=============
•	Flask (Python)<br/>
•	Mysql<br/>
•	Bootstrap<br/>
•	Jquery<br/>
•	Panoramio API<br/>
•	Facebook Graph API (just for login)<br/>
•	REST Architecture<br/>
