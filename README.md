social-mini
===========

a mini social networking site around panoramio. A user can register and login through facebook and find photos anywhere in the world and comment on a photo

features
========

Streams
=======
o	 Recent
o	 Popular
o	 Nearby  
o	 Trending 
o	 User Activity   
o	 My activity 
User (Subjective)
=================
o	Like
o	Flag
o	Comment
o	Tag cloud
o	Suggestions



Technologies 
=============
•	Flask (Python)
•	Mysql
•	Bootstrap
•	Jquery
•	Panoramio API
•	Facebook Graph API (just for login)
•	REST Architecture
